# Employee Management System (Core OOP) 🏗️

This project is a comprehensive demonstration of **Object-Oriented Programming (OOP)** principles in C#. It showcases how to design a modular, scalable, and maintainable system using advanced architectural concepts.

## 🛠️ Key OOP Concepts Applied
* **Abstraction:** Defined an `abstract Employee` base class to enforce a strict template for specialized roles.
* **Inheritance & Polymorphism:** Created specific classes like `Developer`, `Manager`, and `Intern` that inherit common traits while overriding unique behaviors.
* **Interfaces:** Used `IReportable` and `ITrainable` to decouple specific capabilities from the class hierarchy, adhering to the **Interface Segregation Principle**.
* **Encapsulation:** Implemented robust data validation within properties to ensure data integrity (e.g., age and name constraints).




## 🚀 Technical Stack
* **Language:** C#
* **Framework:** .NET Console
* **Core Skills:** Inheritance, Interfaces, Abstract Classes, List Management
